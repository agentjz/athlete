# 目录到代码映射

## 当前主目录地图

### 核心执行系统

- `src/agent/`
  - turn、session、checkpoint、verification、acceptance、runtime stats
- `src/orchestrator/`
  - 输入分析、任务规划、进度读取、路由、派发
- `src/tools/`
  - 内建工具与 runtime tool registry
- `src/skills/`
  - skill catalog、matching、loading、workflow guards
- `src/mcp/`
  - 外部能力接入
- `src/control/ledger/`
  - SQLite 控制面仓库
- `src/tasks/` / `src/team/` / `src/background/` / `src/worktrees/`
  - 各控制面实体的实现层

### 宿主运行边界

- `src/host/`
  - 共享 session 入口
  - 共享 turn 入口
  - 共享 host extra tool 注入

### 宿主适配层

- `src/cli/`
  - commander、one-shot、session 命令
- `src/interaction/`
  - 共享交互驱动
- `src/ui/` / `src/shell/cli/`
  - CLI 展示层
- `src/telegram/`
  - Telegram 通道
- `src/weixin/`
  - 微信通道
