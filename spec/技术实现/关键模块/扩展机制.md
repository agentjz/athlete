# 扩展机制

## 当前正式扩展口

1. tools
2. skills
3. MCP
4. host runtime callbacks / extra tools

## 当前要求

- tool 负责动作
- skill 负责 workflow
- MCP 负责外部能力源
- host 负责宿主注入点

## 当前已落地的关键能力

- `mineru-pdf-reading`
- `mineru-image-reading`
- `mineru-doc-reading`
- `mineru-ppt-reading`
- `mineru_pdf_read`
- `mineru_image_read`
- `mineru_doc_read`
- `mineru_ppt_read`
- `web-research`
- `browser-automation`
- Telegram / 微信 send file tool 注入

## 当前硬约束

1. 不把扩展继续塞回 system prompt。
2. tool / skill / MCP / host 必须通过正式扩展口接入。
3. 宿主 extra tool 必须经 `src/host/` 注入。
