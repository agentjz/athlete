# 命令行产品面

## 当前启动链

- fast path
- lightweight path
- full runtime path

## 当前产品目标

- `--version`、`help`、`config path` 快速返回
- 配置坏了时仍给出修复入口
- full runtime 命令才拉起重模块

## 当前关键文件

- `src/cli.ts`
- `src/cli/program.ts`
- `src/cli/commands/`
- `src/cli/runtime.ts`
- `src/cli/oneShot.ts`

## 当前约束

1. CLI 产品面不定义核心真相源。
2. one-shot 和 interactive 都必须走 `src/host/`。
3. 顶层 CLI 入口不能重新长成总控大文件。
