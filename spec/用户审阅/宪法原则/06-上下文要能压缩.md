# P06 上下文要能压缩

## 原则

上下文一定会满，所以必须有压缩和续跑机制。

## 为什么

如果没有压缩，长任务跑着跑着就会直接死掉。

## 在 Deadmouse 里的含义

- request context 自动压缩
- tool step 到上限后自动 continuation
- teammate loop 可以在多轮里继续同一任务

这条原则直接保护 Deadmouse 当前最强的“耐跑能力”。

- `src/agent/context.ts`
- `src/agent/turn/managed.ts`
- `src/agent/retryPolicy.ts`
