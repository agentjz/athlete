# P01 一个循环一个智能体

## 原则

一个智能体，核心就是一个主循环。

这个循环只做三件事：

- 看上下文
- 决定调用工具还是直接回答
- 把结果接回循环继续

## 为什么

这样结构最简单，最容易继续跑，也最容易查问题。

## 在 Deadmouse 里的含义

Deadmouse 允许有多个 agent loop：

- lead
- teammate
- subagent

但每个 agent 自己都必须有清晰 loop，不能多入口乱跳。

- `src/agent/runTurn.ts`
- `src/execution/worker.ts`
