# P02 加一个工具只加一个处理器

## 原则

主循环不要因为新工具增加而不断改动。

新工具应该只是：

- 增加一个工具文件
- 注册进工具表

## 为什么

这样工具系统才可扩展，不会越加越乱。

## 在 Deadmouse 里的含义

Deadmouse 当前把这条原则落实成：

- 工具定义统一注册
- mode 决定工具集
- MCP 工具也走统一 registry

## 当前对应

- `src/tools/registry.ts`
- `src/tools/runtimeRegistry.ts`
- `src/mcp/registryIntegration.ts`
