# P05 知识按需加载

## 原则

不要一开始把所有知识都塞进 system prompt。

需要时再加载对应 skill。

## 为什么

这样更省上下文，也更接近真正的“按需用知识”。

## 在 Athlete 里的含义

skills 是标准扩展口之一。

当前阶段的重点不是疯狂增加 skill 数量，而是先把：

- skill 怎么发现
- skill 怎么声明用途
- skill 怎么按需加载
- skill 怎么和角色、工具、任务配合

做稳。
