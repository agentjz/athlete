# 本地命令与流程

## 常用命令

```powershell
npm install
npm run build
npm run typecheck
npm test
npm run dev
```

## 推荐开发流程

1. 先确认改动落在哪一层
2. 先补或设计测试
3. 再改实现
4. 运行 `npm test`
5. 最后更新 SPEC / docs

## 写 SPEC 的流程

1. 先写当前真实能力
2. 再写未来方向
3. 检查有没有把两者混在一起
4. 检查单文件是否过胖

## 写代码的流程

1. 先找真相源
2. 再找工具入口
3. 最后再看 UI 是否要同步
